事例1. 天井のシミ→屋根板金劣化

横浜市青葉区 T様 / 築12年 / 緊急対応

68万円

症状

2階天井にコーヒー色のシミ。雨の日にポタポタ音。

原因

屋根の棟板金の釘が浮いて隙間から雨水侵入。

工事内容

棟板金交換・下地木材補修・防水シート貼り直し。工期3日。

事例2. 外壁のひび割れ→構造クラック

川崎市麻生区 M様 / 築15年 / 訪問販売で指摘

85万円

症状

外壁に縦方向の大きなひび割れ。訪問販売で300万円の見積もり。

原因

地震による構造クラック。塗装だけでなく構造補修が必要。

工事内容

構造補修・エポキシ樹脂注入・外壁塗装。工期7日。

事例3. ベランダからの浸水→防水劣化

相模原市南区 K様 / 築18年 / オンライン診断

52万円

症状

ベランダ下の部屋の天井にシミ。雨の翌日に濡れる。

原因

ベランダ防水層の劣化。排水口周りから水が浸入。

工事内容

防水層の全面やり直し・排水口交換。工期4日。

事例4. 雨樋の破損→オーバーフロー

町田市 Y様 / 築10年 / 緊急対応

38万円

症状

外壁がびしょ濡れ。雨樋から滝のように水が溢れる。

原因

雨樋の詰まりと破損。落ち葉が原因で雨水が逆流。

工事内容

雨樋全交換・落ち葉除けネット設置。工期2日。

事例6. 屋根瓦のズレ→台風被害

相模原市緑区 N様 / 築20年 / 緊急対応

95万円

症状

台風後、天井から大量の水。屋根瓦が数枚落ちていた。

原因

台風で瓦が飛散。下地の防水シートも破れていた。

工事内容

瓦の全面点検・補修・防水シート張り替え。工期5日。

事例8. 屋根の谷部分→錆び穴

120万円

症状

天井のシミが徐々に拡大。訪問販売で屋根全交換350万円の見積もり。
