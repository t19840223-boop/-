# 究極の雨漏りLP戦略書

## 🎯 ユーザー心理分析

### 雨漏り発生時の心理フロー

```
【発見】雨漏り発見 → 【不安】家が壊れる？ → 【調査】ネットで検索
    ↓
【比較】どこに頼む？ → 【決断】信頼できる？ → 【行動】問い合わせ
```

### ユーザーが最初に気になること

1. **今すぐ止まるのか？**（緊急性）
2. **いくらかかるのか？**（費用）
3. **信頼できる業者か？**（安心）
4. **どのくらいで来てくれる？**（対応速度）
5. **家が崩れないか？**（安全性）

## 👥 ターゲットペルソナ

### ペルソナ1: 緊急対応層（最優先）
- **年齢**: 40〜60代
- **状況**: 雨漏りが発生中、今すぐ止めたい
- **心理**: パニック、不安、早く解決したい
- **検索**: 「雨漏り 緊急」「雨漏り すぐ」「雨漏り 町田」
- **ニーズ**: 即日対応、24時間対応、明確な料金
- **CVポイント**: 電話ボタン、緊急対応フォーム

### ペルソナ2: 予防検討層
- **年齢**: 40〜60代（子供世代が調査）
- **状況**: 天井にシミ、訪問販売に指摘された
- **心理**: 不安だけどまだ大丈夫？、騙されたくない
- **検索**: 「雨漏り 症状」「外壁 劣化」「訪問販売 嘘」
- **ニーズ**: 無料診断、オンライン相談、信頼性
- **CVポイント**: オンライン診断フォーム

### ペルソナ3: 放置後悔層
- **年齢**: 50〜70代
- **状況**: 気づいていたのに放置、悪化した
- **心理**: 後悔、自己嫌悪、早く直したい
- **検索**: 「雨漏り 放置」「雨漏り 被害」「雨漏り 修理費用」
- **ニーズ**: 誠実な対応、追加費用の透明性
- **CVポイント**: 失敗事例→共感→診断

### ペルソナ4: 子供世代（実は重要）
- **年齢**: 30〜50代
- **状況**: 親の家の雨漏りを心配、代わりに調査
- **心理**: 親を守りたい、悪徳業者を避けたい
- **検索**: 「雨漏り 業者 選び方」「雨漏り 相場」
- **ニーズ**: オンライン完結、透明性、安心感
- **CVポイント**: オンライン診断、メール報告

## 🧠 購買心理の4ステップ

### 1. 失敗（恐怖）
**訴求**: 「放置すると家が崩れる」「修理費が10倍に」
- 実際の崩壊事例
- 費用比較（今：30万円 vs 放置後：300万円）
- タイムリミット訴求

### 2. 共感（理解）
**訴求**: 「私も同じでした」「気づいていたのに…」
- お客様の声（実話）
- 「訪問販売を断って後悔」
- 「親の家が心配で…」

### 3. 気付き（解決策）
**訴求**: 「オンライン診断なら安心」「緊急対応も可能」
- オンライン診断のメリット
- 緊急対応の流れ
- 透明な料金体系

### 4. 実行（行動）
**訴求**: 「今すぐ無料診断」「緊急は電話で即対応」
- 大きなCTAボタン
- 行動障壁の除去
- 限定感（残り◯名）

## 📱 サービス構造

### オンライン診断（基本・無料）
1. 写真アップロード（スマホで撮影）
2. 簡単な質問に回答
3. 24時間以内に診断結果をメール
4. オンライン相談予約（Zoom/LINE）
5. 見積もり作成（無料）

### 緊急対応（有料だが工事で相殺）
1. 電話で即受付（24時間）
2. 最短1時間で現地到着
3. 応急処置（防水シート等）
4. **費用**: 出張費◯◯円 + 応急処置◯◯円
5. **特典**: 本工事契約時は全額値引き

## 🆚 相見積もりサイト批判戦略

### 相見積もりサイトのデメリット（ユーザー視点）

#### ❌ デメリット1: 個人情報の大量流出
> 「一括見積もりサイトに登録したら、10社以上から営業電話が…」
- 個人情報が複数業者に流れる
- 営業電話・メールが止まらない
- 本当に信頼できる業者かわからない

#### ❌ デメリット2: 価格競争の弊害
> 「安すぎる見積もりには裏がある」
- 業者が価格を叩き合う
- 利益を削るため手抜き工事のリスク
- 下請け・孫請けの連鎖で品質低下

#### ❌ デメリット3: 優良店の嘘
> 「『優良店』と書いてあるが、実は訪問販売会社」
- 掲載基準が不透明
- 高額な掲載料を払える業者が優先
- 訪問販売会社も多数登録

### ✅ 当社の直接依頼のメリット

#### ✅ メリット1: 個人情報保護
- 当社のみで情報管理
- 営業電話は一切なし
- オンラインで完結可能

#### ✅ メリット2: 適正価格と品質
- 中間マージンなし
- 適正価格で高品質
- 自社施工で責任施工

#### ✅ メリット3: 透明性
- 毎日メール報告
- 写真付き進捗報告
- 追加費用は事前相談
